## 基础定义

### E2 (17β-雌二醇)

| 属性 | 说明 |
|------|------|
| **本质** | 人体卵巢分泌的最主要、活性最强的天然雌激素 |
| **形态** | 生物学终产物，直接与雌激素受体结合产生效应 |
| **口服问题** | 经肝脏首过效应代谢，生物利用度极低 |

### 戊酸雌二醇 (EV)

| 属性 | 说明 |
|------|------|
| **本质** | 雌二醇的前体药物（Prodrug） |
| **结构** | E2 + 戊酸基团（第17位碳原子） |
| **原理** | 体内酯酶切断戊酸链，还原为 E2 |
| **特点** | 无活性，需转化后才起效 |

## 核心对比

| 维度 | E2 (17β-雌二醇) | 戊酸雌二醇 (EV) |
|------|-----------------|-----------------|
| **生物活性** | 极高（直接起效） | 无活性（需水解转化） |
| **半衰期** | 很短（代谢极快） | 较长（长链减缓降解） |
| **常见剂型** | 凝胶、贴剂、喷雾 | 口服片剂（补佳乐）、长效油剂注射 |
| **肝脏代谢** | 口服损耗巨大 | 口服稳定性较好 |
| **吸收效率** | 经皮吸收效果极佳 | 消化道或肌肉注射吸收 |

## 为什么戊酸雌二醇更常用？

### 更平稳的血药浓度

```
纯 E2 口服 → 高峰值 → 迅速跌落（波动大）
EV 口服   → 限流阀效应 → 释放平缓（波动小）
```

### 更好的口服耐受性

- 对肝脏的瞬时冲击比纯口服 E2 温和
- 维持更平稳的血药浓度

### 长效潜力

- 油剂注射液（如 Delestrogen）
- 在肌肉组织形成"储库"
- 缓慢释放，维持 5-7 天

## 如何选择？

{{< callout info >}}
**核心原则：口服选 EV，透皮选 E2**
{{< /callout >}}

### 给药方式对比

| 给药方式 | 推荐选择 | 说明 |
|----------|----------|------|
| **口服** | 戊酸雌二醇 (EV) ✅ | 纯 E2 口服几乎无效 |
| **经皮（凝胶/贴剂）** | 半水合物雌二醇 (E2) ✅ | 避开肝脏首过代谢 |
| **肌肉注射** | EV 或环戊丙酸雌二醇 ✅ | 维持长时间药效 |

### 临床推荐

| 场景 | 推荐方案 |
|------|----------|
| 日常 HRT（首选） | **经皮雌二醇凝胶** — 对肝脏和血管影响最小 |
| 无法使用经皮 | 口服 EV（补佳乐） |
| 需要长效 | EV 油剂注射（Delestrogen） |

## ⚠️ 血栓风险对比

| 给药方式 | 血栓风险 | 原因 |
|----------|----------|------|
| 口服 E2 | ⚠️⚠️⚠️ 高 | 肝脏首过效应，影响凝血因子 |
| 口服 EV | ⚠️⚠️ 中 | 仍有一定影响 |
| **经皮 E2** | ✅ 低 | 避开肝脏，不影响凝血 |

{{< callout warning >}}
**首选透皮给药** — 血栓风险最低

在达到相同 E2 水平的前提下，经皮雌二醇凝胶对肝脏和血管的影响最小。
{{< /callout >}}

## 总结

| 结论 | 说明 |
|------|------|
| EV 是 E2 的前体 | 需转化才起效 |
| 口服选 EV | 纯 E2 口服无效 |
| 经皮选 E2 | 避开肝脏代谢 |
| 注射选 EV | 长效稳定 |
| **首选透皮** | 血栓风险最低 |
