## 📊 BMI 分级

### 什么是 BMI？

BMI（Body Mass Index，身体质量指数）是衡量体重与身高比例的指标。

**计算公式：**
```
BMI = 体重(kg) ÷ 身高²(m)
```

**BMI 分级（中国标准）：**

| BMI | 分类 | 建议 |
|-----|------|------|
| < 18.5 | 体重不足 | 增重 |
| 18.5 - 23.9 | 正常 | 保持 |
| 24 - 27.9 | 超重 | 减重 |
| ≥ 28 | 肥胖 | 减重 |

## 🧠 心理建设

### 减肥的本质

**核心认知：** 减肥的本质不是你变瘦了，**本质是你的生活习惯变了**。

瘦下来代表着：
1. 少喝或不喝奶茶（所有市售饮料尽可能不喝）
2. 限量吃炸鸡薯片薯条类油炸食品
3. 碳酸饮料换成无糖，酸奶换成纯牛奶
4. 能喝水的尽量喝水

瘦下来不代表：
1. 每天要饿肚子（你每天都能吃饱）
2. 每天吃难吃的东西
3. 大量痛苦的锻炼

---

### 三个关键认知

#### 1. 皮肤更新需要时间

皮肤是花了许许多多年才变黑/变差的，你同样需要持之以恒地去调节、恢复。

**时间参考：**
- 日晒、饮食、作息导致的皮肤变差：至少 20 天才能看到细微效果
- 脚部等精细部位：两到三年后才能明显改善

#### 2. 没有万能药

护肤品不是万能的。精致的皮肤 = 护肤品 + 防晒 + 美容措施 + 饮食规律 + 作息规律 + 良好心态。

#### 3. 最贵的 ≠ 最好的

适合自己才最重要。别人用得好 ≠ 你用得好。

---

### 减肥成功要点

| 要点 | 说明 |
|------|------|
| **睡够** | 睡眠不足会导致代谢下降 |
| **心情好** | 压力大会影响激素，导致脂肪堆积 |
| **坚持** | 给身体时间适应新习惯 |
| **不要急** | 体重每周减 0.5-1kg 是理想速度 |

## 🎯 行动指南

### 第一步：评估当前状态

1. 计算自己的 BMI
2. 记录当前饮食习惯
3. 设定合理目标（不要一下定太大）

### 第二步：调整饮食结构

1. 计算目标体重的三大营养素摄入量
2. 开始用厨房秤记录食物重量
3. 减少精制碳水（馒头、奶茶、薯片）

### 第三步：建立习惯

1. 固定吃饭时间
2. 记录每日饮食
3. 每周称一次体重

### 第四步：坚持下去

1. 给自己 3 个月时间
2. 不要因为偶尔放纵就放弃
3. 记住：减肥是生活方式的改变，不是短期冲刺
