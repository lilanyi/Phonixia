## 雌激素类型

### E2 (17β-雌二醇)

| 属性 | 说明 |
|------|------|
| **本质** | 人体天然雌激素 |
| **活性** | 直接起效 |
| **剂型** | 凝胶、贴片、喷雾、舌下片 |
| **推荐度** | ⭐⭐⭐⭐⭐（透皮） |

### EV (戊酸雌二醇)

| 属性 | 说明 |
|------|------|
| **本质** | E2 的前体药物 |
| **活性** | 需体内转化 |
| **剂型** | 口服片剂（补佳乐）、注射油剂 |
| **推荐度** | ⭐⭐⭐⭐（口服首选） |

## 给药方式对比

| 方式 | 生物利用度 | 稳定性 | 血栓风险 | 推荐度 |
|------|------------|--------|----------|--------|
| **透皮凝胶** | 高 | 稳定 | 低 ✅ | ⭐⭐⭐⭐⭐ |
| **透皮贴片** | 高 | 最稳定 | 低 ✅ | ⭐⭐⭐⭐⭐ |
| **舌下含服** | 高 | 波动大 | 中 | ⭐⭐⭐⭐ |
| **口服 EV** | 中等 | 中等 | 中 ⚠️ | ⭐⭐⭐⭐ |
| **口服 E2** | 低 | 波动大 | 高 ⚠️ | ⭐⭐ |
| **注射** | 最高 | 周期性 | 中 | ⭐⭐⭐⭐ |

{{< callout info >}}
**首选透皮给药**：血栓风险最低，对肝脏影响最小。
{{< /callout >}}

## 抗雄激素类型

### 螺内酯 (Spironolactone)

| 属性 | 说明 |
|------|------|
| **机制** | 阻断雄激素受体 |
| **优点** | 便宜、易获取 |
| **缺点** | 高血钾风险 |
| **监测** | 血钾每 3 月 |

### CPA (色普龙)

| 属性 | 说明 |
|------|------|
| **机制** | 抑制睾酮生成 |
| **优点** | 效果强 |
| **缺点** | 需处方 |
| **风险** | 低剂量时风险低 |

### GnRH 激动剂

| 属性 | 说明 |
|------|------|
| **机制** | 完全抑制性腺 |
| **优点** | 效果最强 |
| **缺点** | 昂贵 |
| **风险** | 骨质疏松（需配合雌激素） |

## 黄体酮

### 是否必要？

{{< callout info >}}
**黄体酮不必需**，但多数人报告有帮助。
{{< /callout >}}

### 作用

| 作用 | 效果 |
|------|------|
| 促进乳腺发育 | 明显 |
| 改善睡眠 | 明显 |
| 情绪调节 | 因人而异 |

### 副作用（口服）

| 副作用 | 原因 | 解决方案 |
|--------|------|----------|
| 焦虑 | 肝脏代谢产物 | 改用直肠给药 |
| 情绪波动 | 肝脏代谢 | 逐渐加量 |

{{< callout info >}}
**建议直肠给药**：避免肝脏代谢，减少副作用。
{{< /callout >}}

## 药物选择建议

### 初学者

| 优先 | 选择 |
|------|------|
| 雌激素 | 透皮凝胶或口服 EV |
| 抗雄 | 螺内酯（便宜易获取） |

### 效果优化

| 优先 | 选择 |
|------|------|
| 雌激素 | 注射或透皮凝胶 |
| 抗雄 | CPA 或 GnRH 激动剂 |
| 辅助 | 黄体酮（6 月后添加） |

### 安全优先

| 优先 | 选择 |
|------|------|
| 雌激素 | 透皮凝胶（血栓风险最低） |
| 抗雄 | 螺内酯 + 定期监测 |
