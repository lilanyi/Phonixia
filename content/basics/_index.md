## 本节内容

### 什么是 HRT？

激素替代治疗（Hormone Replacement Therapy, HRT）是跨性别女性实现身体女性化的主要医疗手段。

### 核心概念

| 概念 | 说明 |
|------|------|
| **雌二醇 (E2)** | 主要女性激素，促进乳房发育、脂肪分布女性化 |
| **睾酮 (T)** | 男性激素，需要抑制到女性范围 |
| **抗雄激素** | 抑制睾酮的药物（螺内酯、CPA、GnRH 激动剂等） |
| **黄体酮** | 辅助激素，可能促进乳房发育 |

### 给药方式

| 方式 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **透皮凝胶** | 吸收好、血栓风险低 | 需要每天涂抹 | ⭐⭐⭐⭐⭐ |
| **透皮贴片** | 方便、稳定 | 成本较高 | ⭐⭐⭐⭐ |
| **舌下含服** | 高生物利用度 | 波动大 | ⭐⭐⭐⭐ |
| **口服** | 方便、便宜 | 血栓风险 | ⭐⭐⭐ |
| **注射** | 效果好、稳定 | 需要技术 | ⭐⭐⭐⭐ |
