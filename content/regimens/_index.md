## 本节内容

### 标准方案类型

| 方案类型 | 适用人群 | 特点 |
|----------|----------|------|
| **口服 EV + 抗雄** | 初学者 | 方便、便宜 |
| **透皮凝胶 + 抗雄** | 安全优先 | 血栓风险最低 |
| **舌下含服 + 抗雄** | 效果优先 | 高生物利用度 |
| **注射 EV** | 长期稳定 | 效果好、需技术 |

### 方案选择决策

```
开始 HRT
    │
    ├─ 血栓风险低？
    │     ├─ 是 → 口服 EV（便宜方便）
    │     └─ 否 → 透皮凝胶（最安全）
    │
    └─ 需要长期稳定？
          ├─ 是 → 注射 EV
          └─ 否 → 口服/透皮
```

## 高风险药物警告

| 药物 | 风险 | 建议 |
|------|------|------|
| **炔雌醇** | 血栓、肝癌 | 🚫 绝对避免 |
| **合成孕激素** | 肝脏毒性、脑膜瘤 | 🚫 避免使用 |
| **GnRH 激动剂** | 骨质疏松 | ⚠️ 需监测骨密度 |
| **螺内酯** | 高血钾 | ⚠️ 定期检测血钾 |
